# health center website QA Project

This project shows manual testing for a health center website .

## Files included
- **Test Scenarios** – what to test
- **Test Cases** – step-by-step instructions
- **Bug Reports** – issues found

## Skills demonstrated
- Functional Testing
- UI Testing
- Responsive Testing
- Positive and Negative Testing

## How to use
1. Open the health center app in browser
2. Follow the Test Cases
3. Check Bug Reports for issues
